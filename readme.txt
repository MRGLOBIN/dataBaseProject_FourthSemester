this project is created on node.js react and mongoDB

to run this project please install node.js nad mongoDB

now follow these steps to run the project
 1. open front-end folder in terminal and use npm install
 2. then use npm start

 3. now open another terminal and move inside back-end folder
 4. there again use npm install command
 5. then use npm run setcounter
 6. finally use npm start command

now the server is running and the browser will show the output
